Static Site Rebuild

This project focuses on understanding how real websites are structured from scratch using plain HTML, CSS, and basic JavaScript without relying on external frameworks.

Skills You Will Practice

HTML Structure

Learn how to properly structure a webpage using semantic HTML elements such as headers, sections, articles, and footers to improve readability and accessibility.

CSS Layout & Styling

Practice layout creation using Flexbox, spacing techniques, colors, and responsive units while keeping the design simple and clean.

Responsive Design

Understand how to make layouts adapt for mobile and desktop screens using media queries and flexible containers.

How This Project Works

The Static Site Rebuild project is divided into daily tasks. Each day focuses on a specific part of the website such as header creation, hero section design, content layout, and responsiveness.

Instead of copying designs directly, students rebuild layouts by observing structure and understanding how different sections are connected together.

This approach helps develop problem-solving skills and confidence in building websites independently.

Learning Outcomes

  • Clear understanding of webpage layout flow
  • Better control over spacing, alignment, and responsiveness
  • Ability to build websites without frameworks
  • Improved confidence in frontend fundamentals
  • Hands-on experience with real project structure
